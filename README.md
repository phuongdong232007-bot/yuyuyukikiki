# BEESO BUDGET

Sổ chi tiêu cute cho sinh viên (HTML / CSS / JS).

## Cách chạy

**Chỉ cần mở file `index.html`** trong trình duyệt.

Trên GitHub: bấm vào `index.html` → nút **Raw** hoặc tải về rồi mở bằng Chrome/Edge.

## Có gì bên trong?

1. Trang giới thiệu (sinh viên tiếng Hàn Truyền thông, 02/03/2007)
2. Sổ chi tiêu: nhập → tự gắn category → tính số dư ngày/tuần/tháng
3. Chart + lời khuyên tiêu dùng
4. Đổi ngôn ngữ **VI ↔ KO**
